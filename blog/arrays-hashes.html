Hash or Array?
January 4, 2016

Hashes and arrays are similar in the most general terms of being data structures. In addition, they both include values that include strings, integers, or floats. However, they still have key differences. For instance, brackets [] are used to index into an array, while hashes use braces {} .

Arrays

  • An array stores elements one after the other and can be accessed by index integers.
  • Arrays can be created by using the array class, the initializer, or by adding elements as you go along.
  • The first array element, if one, is located at index[0] and the last element is located at array.length - 1.
  • Arrays can be modified by simply replacing an element with another.
  • Arrays are specifically indexed by integers.
  • The size can of an array can be set when you create an name = Array.new().
  • Example: new_array[0] = "string" new_array[3] = 35

Hashes

  • A hash is a more optimized collection, when compared to an array.
  • Hashes can be created many ways like arrays, but the most common would be calling name = Hash.new().
  • Hashes are unique in that its elements are key-value pairs.
  • Unlike arrays, hashes can be indexed by any object type.
  • Example: new_hash = { "key" => value, "key" => value }
