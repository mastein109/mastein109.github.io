What the Enumerable# ?!
January 11, 2016

Enumberables

The Enumerable module in Ruby is a collection of objects that have many characteristics in common. Further, Enumerable inclue several methods that are similar to each, in that they iterate through the whole collection and return elements from it.

The cycle method

Enumberable#cycle iterates through all elements of a collection in a loop again and again. The cycle method will loop infinitely, unless told otherwise. In order, to stop the infinite looping, an integer can be provided as an argument. See below for a visual example.

In line 1 of the code block, you will see a list of elements (in this case a few Ruby languages) defined. In line 2, the cycle method is called without any arguments. Therefore, the list will be printed infinitely. Finally, in line 3, the cycle method is called with an argument of integer 2. As shown, the list of elements will be printed twice and then end the cycle method.
