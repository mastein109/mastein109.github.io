Ruby Classes

January 18, 2016

Classes are used in Ruby to group together similar objects. Classes include variables that are associated with a specified instance of the class, and methods work with those variables to do something. Classes are defined by using the word "class" followed by a Class_Name. There is one parent class which inherits the methods defined within and can contain a number of child classes.

The best way to demonstrate how a class works is to provide an example. The example I am going to use will allow for my puppy to be fed and taken out based on how she is responding. 


    class Puppy
        def initialize(name)
          @name = name
            puts "#{@name} has awoken!"
          end
    end

    #Create subclass of puppy to define puppy tasks to be done
    class Puppy_Tasks < Puppy

        def feed(hungry)
          @hungry = hungry
          if @hungry == "no"
            puts "#{@name} is full!"
          else @hungry == "yes"
          puts "Feed #{@name} two cups of kibble."
          end
        end

        def walk
          if @hungry == "no"
            puts "Take #{@name} on a walk to go to the bathroom"
          else @hungry == "yes"
            puts "Play fetch with #{@name}"
          end
        end

        def sleepy(tired)
          @tired = tired
          if @tired == "no"
            puts "Walk #{@name} around the block."
          else @tired == "yes"
            puts "#{@name} is a sleepy, snoring baby."
          end
        end
      end


Now that we have defined our Class Puppy and different methods with instance variables, we can test the program out. Here are the commands to call the class and its methods.




  puppy = Puppy_Tasks.new "Charlie"

  puppy.feed("yes")
  puppy.walk
  puppy.feed("no")
  puppy.walk
  puppy.sleepy("yes")
