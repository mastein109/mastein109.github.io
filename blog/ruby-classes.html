Ruby Classes

January 18, 2016

Classes are used in Ruby to group together similar objects. Classes include variables that are associated with a specified instance of the class, and methods work with those variables to do something. Classes are defined by using the word "class" followed by a Class_Name. There is one parent class which inherits the methods defined within and can contain a number of child classes.
